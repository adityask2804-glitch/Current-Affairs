# UPSC Current Affairs - Automated Study Assistant

A **100% FREE** automated app that fetches, summarizes, and organizes current affairs from leading Indian newspapers for UPSC preparation.

## 🎯 Features

- ✅ **Auto-scrapes** editorials from The Hindu & Indian Express daily at 7:30 AM IST
- ✅ **AI Summarization** using OpenAI GPT-5.2 (UPSC-focused, 150-250 words)
- ✅ **Smart Categorization** into 7 UPSC-relevant topics
- ✅ **PDF Export** - Download summaries (all or selected articles)
- ✅ **Android App** - Native mobile experience
- ✅ **Completely FREE** - Deployed on free tier services

## 📱 What This App Does

**Daily at 7:30 AM (Automatic):**
1. Fetches 60 articles from The Hindu & Indian Express
2. Summarizes each with AI (UPSC-focused)
3. Categorizes into 7 topics
4. Stores in database

**Topics:**
- Polity & Governance
- Economy
- International Relations
- Environment & Ecology
- Science & Technology
- Social Issues
- Miscellaneous

## 🚀 Deployed on Render.com

Backend API running on Render.com free tier with MongoDB Atlas.

## 💻 Tech Stack

- **Backend:** FastAPI, MongoDB, APScheduler
- **AI:** OpenAI GPT-5.2 via Emergent LLM
- **Scraping:** newspaper3k, feedparser
- **PDF:** ReportLab
